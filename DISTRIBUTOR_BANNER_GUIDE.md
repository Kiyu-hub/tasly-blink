# Distributor Recruitment Banner Guide

## Overview
This guide explains how to create and manage the "Become a Distributor" banner through the admin panel.

## Creating the Banner

### Step 1: Access Admin Panel
1. Navigate to `/admin-tasly-ghana-346`
2. Enter password: `health2024`
3. Click on the **Ads** tab

### Step 2: Create New Advertisement
Click the "Add New Ad" button and fill in the following details:

#### Banner Details
- **Title**: `Become a Tasly Distributor`
- **Description**: `Join our network of successful distributors and build a thriving business with authentic Tasly products`
- **Image URL**: Use one of these options:
  - Professional banner image URL (recommended size: 1200x400px)
  - Example: `https://images.unsplash.com/photo-1560179707-f14e90ef3623?w=1200&h=400&fit=crop`
  - Or upload your own banner to an image hosting service

#### Positioning
- **Position**: Select `homepage-top` for maximum visibility
- This will display the banner prominently at the top of the homepage

#### Call-to-Action
- **Button Text**: `Apply Now`
- **Button Link**: `/become-distributor`

#### Scheduling (Optional)
- **Start Date**: Leave empty or set to current date
- **End Date**: Leave empty for permanent display
- Check **Active** to enable the banner immediately

### Step 3: Save and Verify
1. Click **Save Ad**
2. Navigate to the homepage to see the banner
3. Click the "Apply Now" button to test the link to the registration form

## Banner Best Practices

### Image Guidelines
- **Dimensions**: 1200x400px (or 3:1 ratio)
- **File Format**: JPG or PNG
- **File Size**: Under 500KB for fast loading
- **Content**: Include text overlay or graphic elements that communicate:
  - "Become a Distributor"
  - "Join Our Network"
  - "Build Your Business"

### Recommended Image Sources
1. **Custom Design**: Create a branded banner using:
   - Canva (free design tool)
   - Adobe Express
   - Figma

2. **Stock Images** (with text overlay):
   - Unsplash: https://unsplash.com/s/photos/business-partnership
   - Pexels: https://www.pexels.com/search/handshake/
   - Pixabay: https://pixabay.com/images/search/business-team/

### Color Scheme
Use Tasly Ghana brand colors:
- Primary Green: #10b981 (or similar health-related green)
- Text: White or dark text for contrast
- Accent: Gold or complementary colors

## Alternative Positions

You can create multiple distributor recruitment ads for different positions:

1. **homepage-top**: Main banner (most visible)
2. **homepage-middle**: Secondary placement
3. **products-sidebar**: Visible while browsing products
4. **checkout-confirmation**: Post-purchase recruitment

## Managing the Banner

### Editing
1. Go to Admin → Ads tab
2. Find the distributor banner in the list
3. Click **Edit** to modify details
4. Click **Save** to apply changes

### Disabling Temporarily
- Uncheck the **Active** toggle in the ad editor
- The banner will be hidden but not deleted

### Scheduling
- Set **Start Date** for future activation
- Set **End Date** for limited-time recruitment drives

## Troubleshooting

### Banner Not Showing
1. Verify **Active** is checked
2. Check **Start Date** is not in the future
3. Check **End Date** has not passed
4. Ensure **Position** is set correctly
5. Clear browser cache

### Image Not Loading
1. Verify image URL is accessible
2. Check image URL starts with `https://`
3. Test URL in a new browser tab
4. Use a different image hosting service if needed

### Link Not Working
1. Verify Button Link is `/become-distributor`
2. Check for typos in the URL
3. Ensure BecomeDistributor page is properly routed
